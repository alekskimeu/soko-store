
# Soko Store

An e-commerce Frontend web application.

## Features

- Products list
- Product details
- Checkout

  
## Tech Stack

React, CSS, Material UI icons


  
## Run Locally

Clone the project

```bash
  git clone https://github.com/alekskimeu/soko-store-front.git
```

Go to the project directory

```bash
  cd soko-store-front
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
